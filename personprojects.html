Epic Forums

  Over the Summer of 2025, I made a website called Epic Forum. It is a gaming-centric forum website where users can make posts and share images and videos about their favorite games. Here, you can make paragraph-sized posts, upload a video, upload a screenshot, or reply to any previous post of any kind. In addition, this website uses a user/password based system, where users can register to make an account and need to login to create a post or reply to a post. To create this project, I used Django/Python for the backend and Database, HTML for the frontend, and css for the design. I did this entirely in Visual Studio Code.

Hosted on: monkeywizard38.pythonanywhere.com
